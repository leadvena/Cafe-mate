# CafeMate Design System

## Palette (OKLCH based)
- **Cream**: `oklch(0.98 0.01 70)` (Background)
- **Brown Dark**: `oklch(0.25 0.05 40)` (Text Primary, Buttons)
- **Brown Mid**: `oklch(0.45 0.1 40)` (Accents, Hover states)
- **Green Accent**: `oklch(0.55 0.08 150)` (Success, Highlights)
- **Text Primary**: `oklch(0.2 0 0)` (Deep black/brown)
- **Text Muted**: `oklch(0.5 0.02 0)` (Soft grey/brown)

## Typography
- **Headings**: `Playfair Display` (Serif) - Editorial, elegant.
- **Body**: `Inter` (Sans) - Functional, readable.
- **Scale**: 8px grid based modular scale.

## Spacing
- **Base**: 8px
- **Sections**: 80px - 128px vertical padding.
- **Cards**: 24px - 32px internal padding.

## Components
- **Buttons**: Rounded-full, high contrast, tactile press effect.
- **Cards**: Minimal borders, subtle shadows, large padding.
- **Inputs**: Clean, focus states with soft ring and border transitions.

## Motion
- **Transitions**: 200ms ease-out (exponential).
- **Interactions**: Subtle scale transforms (0.97) on click.
- **Entrance**: Staggered fade-ins for list items.
